Trainer — Log → Vault Migration

Parses your plain-text daily log and writes one YYYY-MM-DD.md file per day into References/Trainer/ inside your vault folder.

Step 1
Paste your log
Paste the full log text. Date lines like 4/5 or 4/5 50u 240 start each day.
Step 2
Pick your vault folder
Select the root folder of your vault. The migration writes into References/Trainer/ inside it.
Step 3
Write files
Writes one YYYY-MM-DD.md per day. Existing files are overwritten.
